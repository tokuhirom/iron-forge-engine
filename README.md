# 鉄塊機関 ―IRON FORGE ENGINE―

## 概要

スチームパンクの世界を舞台にした、スマートフォン向けシューティングパズルゲーム。

---

## 世界観

薄暗い蒸気工場。空からスクラップが降り注ぐ中、プレイヤーは蒸気砲台を操り、
鉄塊を整形して製品として出荷し続ける。歯車が回り、蒸気が噴き出す工場を守れ。

---

## ゲームルール

### 基本フロー

1. 上からスクラップ（鉄塊）が降ってくる
2. プレイヤーは蒸気砲台を左右に動かしながら玉を撃つ
3. 玉がスクラップに当たるとスクラップが横に伸びる
4. スクラップが矩形（長方形）を形成したら製品として出荷される（消える）
5. スクラップが画面下まで到達したらゲームオーバー

### スコア

- 矩形完成時に出荷スコアが入る
- 矩形が大きいほど高スコア
- 連続出荷（連鎖）でボーナス

---

## 操作体系（スマートフォン）

| 操作 | アクション |
|------|-----------|
| 画面をスワイプ（左右） | 蒸気砲台を移動（相対移動） |
| 画面をタップ | 玉を射出 |

### 設計方針

- スワイプと射出は独立した操作として扱う
- スワイプ量に比例して自機が移動する（相対移動）
- タップで即時射出

---

## ビジュアルイメージ

- **スタイル**: レトロ・スチームパンク
- **自機**: 歯車や蒸気管がついた砲台
- **弾**: 鉄球・蒸気弾
- **ブロック**: 鉄塊・スクラップ
- **背景**: 薄暗い工場、煙突の煙、回転する歯車
- **エフェクト**: 矩形完成時に蒸気と火花、出荷演出
